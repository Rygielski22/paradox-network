# Paradox Network

**Premium Bedrock Proxy**  
Dark purple. White. Animated. Polished.

## Features

- Clean MITM proxy for Minecraft Bedrock (Lifeboat 1.26.30)
- Stunning terminal banner in dark purple + pure white
- Animated arraylist (gradient purple → white)
- ClickGUI (expandable)
- Movement: Fly, Velocity
- Combat: Killaura foundation
- Themed chat commands with consistent branding
- Easy module system

## Theme

- **Primary**: §5 (dark purple)
- **Accent**: §d + §f (white)
- Terminal: ANSI 35 / 95 purple, bright white

## Quick Start

```bash
npm install
npm start
```

Connect to `127.0.0.1:19132`

Commands:
- `?help`
- `?fly`
- `?vel`
- `?killaura`
- `?clickgui`
